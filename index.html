Capabilities

Everything on one screen.

Dense, readable, accurate. The data a pilot actually wants, without touching a web app.

◆
Flight Category Color-Coding

Instant visual status matching FAA standard flight rules. No guessing.

VFR MVFR IFR LIFR
⊕
Wind Rose Visualization

ASCII compass rose with directional arrow and center dot. Wind at a glance, numbers below.

▇
6-Hour Sparklines

Unicode trend bars for temperature, wind speed, and altimeter. See the trend, not just the number.

◌
TAF Forecasts

Toggle Terminal Aerodrome Forecasts alongside the current observation. Decoded, formatted.

⌦
Interactive Mode

Full-screen live dashboard built on prompt_toolkit. Keyboard shortcuts for everything.

≡
Multi-Station

Fetch weather for multiple ICAO stations in a single command. Compare conditions instantly.

△
Density Altitude

Computed locally from temp and QNH via ISA lapse rate. Always shown, never hidden.

○
No API Key Required

Data from Aviation Weather Center (NOAA) and Iowa State Mesonet. Free. No account needed.

Usage

Simple syntax.

One command gets you the weather. Options handle everything else.

Syntax
# Default station
$ metar

# Specific or multiple stations
$ metar KJFK
$ metar KJFK KLGA KEWR

# Options
$ metar --taf               # include TAF
$ metar --raw               # raw METAR string
$ metar -i                  # interactive mode

# Save your home airport
$ metar --set-default MMML
Examples
# Check home airport (default station)
$ metar

# Pre-flight: destination + alternate + TAF
$ metar KLAX KBUR --taf

# Raw string only, for logging
$ metar KJFK --raw

# Set Mexicali as home station
$ metar --set-default MMML

# Live updating dashboard
$ metar --interactive
Interactive Mode

Full-screen dashboard.

A live weather panel that stays open and responds to keyboard commands.

Launch once, navigate with the keyboard. Toggle TAF, inspect the raw string, switch stations, refresh on demand — all without leaving the terminal.

$ metar -i
tToggle TAF forecast
rShow / hide raw METAR string
uRefresh — re-fetch live data
sSearch a new ICAO station
cSet default station interactively
qQuit
Installation

Up in 10 seconds.

Requires Python 3.10+. pipx keeps it isolated from your system packages.

01
Install via pipx

Recommended — fully isolated, available everywhere in your shell.

pipx install metar-cli
02
Set your home airport

Saved to ~/.config/metar/config. Override anytime with $METAR_ICAO.

metar --set-default KJFK
03
Check the weather

That's it. Works anywhere in your terminal.

metar --taf
# From source (development)
$ git clone https://github.com/alexgc96/metar-cli.git
$ cd metar-cli
$ python3 -m venv .venv && source .venv/bin/activate
$ pip install -e .
